# Financial Health Score Calculator

## Overview
This project is a financial health scoring system that analyzes a user’s financial condition based on income, expenses, savings, debt, and investments. It generates a financial score between 0 and 100.

## Features
- Income and expense tracking
- Savings analysis
- Debt ratio calculation
- Investment tracking
- Composite financial score generation
- Graphical representation of score

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript
- React.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## How it Works
1. User enters financial data.
2. System processes the data.
3. Different financial metrics are calculated.
4. A final health score is generated.
5. Results are shown in graphs and charts.
